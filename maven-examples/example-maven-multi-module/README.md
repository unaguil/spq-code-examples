Multi-module Maven Project Example
==================================

This project shows a multi-module project composed of a Parent POM (also called Super POM) and two sub-modules: one containing the main web application and the other a library required by the web application as a dependency.

Compiling the project

    mvn compile

Launching the tests

    mvn test

Create the distributable version of all modules (JAR/WAR)

    mvn package

The packaged version of `my-app` will be written to *my-app/target* and can be run by adding the generated JAR to the classpath and specifying the main class

    java -cp my-app/target/my-app-1.0-SNAPSHOT.jar com.mycompany.app.App

To delete and clean all the artifacts generated by previous builds, run the command

    mvn clean

and check that the *target* directories in each module are removed.
