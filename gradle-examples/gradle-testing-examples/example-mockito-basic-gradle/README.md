Gradle Mockito Basic
=====================

This project contains a basic Gradle skeleton for the Mockito example used in
the testing samples.

Compiling the project

    ./gradlew compileJava

Launching the tests

    ./gradlew test

Create the distributable JAR

    ./gradlew build

Generate Javadoc

    ./gradlew javadoc

Clean build artifacts

    ./gradlew clean
